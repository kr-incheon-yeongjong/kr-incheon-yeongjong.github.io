# 영종구 · 인천광역시

> 영종구 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-incheon-yeongjong.github.io/**

영종구는 2026년 7월 1일 옛 중구의 섬 지역인 영종도·용유도·무의도 일원이 분리되어 신설된 인천광역시의 자치구다. 인천국제공항이 자리한 관문 도시로, 바다를 경계로 내륙권과 생활권이 분리되어 있던 중구 도서부가 하나의 구로 독립했다. 관할 면적은 약 125.82㎢, 인구는 2025년 기준 약 13만 7천여 명으로 영종하늘도시·운서·운남 일대를 중심으로 빠르게 성장하는 신흥 도시다. 행정동은 영종동·영종1동·영종2동·운서1동·운서2동·용유동으로 구성되며, 임시 구청사는 중산동 하늘중앙로에 두고 있다. 인천경제자유구역(IFEZ) 영종지구를 품어 공항·물류·관광·MICE 기능이 집약된 국제도시이자, 을왕리·마시안 해변과 무의도 등 서해의 대표적 해양 관광지를 함께 아우른다.
